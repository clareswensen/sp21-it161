Welcome

This site demonstrates an array of html structures and CSS styles. Research examples pages include Social Media Optimization (SMO), Web Accessibility, and the use of Forms. Drop down navagation, image galleries, live webcams, and use of forms have been demostrated on this site.

Web Development is a dynamic field and requires constant learning. By looking to online resources, such as w3schools.com and whatwg.com, a developer can stay up to date with the emerging standards. I plan on using the inspector tool in the browser to view the approach well established sites have taken in their code. Another way to expand my knowledge, is to participate in the open source oppurtunities on github and utilize it as a commmunity resource.
